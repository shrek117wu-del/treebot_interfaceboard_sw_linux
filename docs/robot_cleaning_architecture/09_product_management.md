## 1. 产品概述

### 1.1 产品定义

**轮臂式厕所清洁机器人**是一款用于商用卫生间的全自主清洁机器人，融合移动底盘导航与6自由度机械臂操作，通过 VLA（Vision-Language-Action）模型驱动，实现洗手台、马桶、小便池、地面、墙面的自主清洁。

### 1.2 核心价值主张

| 价值点 | 描述 |
|--------|------|
| 替代人力 | 解放保洁工人，解决用工难、成本高问题 |
| 全天候运营 | 7×24 小时自主清洁，无需人工排班 |
| 标准化质量 | 消除人工清洁质量不稳定问题 |
| 数据可视化 | 清洁记录、设备状态实时上报云平台 |
| VLA 智能化 | 自然语言下发任务，系统自适应清洁策略 |

### 1.3 目标市场与场景

| 应用场景 | 优先级 |
|---------|--------|
| 机场、高铁站、交通枢纽卫生间 | P0（首发场景） |
| 大型商场、购物中心 | P1 |
| 医院、医疗机构 | P1 |
| 酒店、写字楼 | P2 |
| 学校、政府机关 | P2 |
